NAME

podcatcher-arguments-format - podcatcher arguments file format

DESCRIPTION

Files specified with the podcatcher --arguments option are text files that are formatted as follows:

  • Each line contains no more than one argument.
  • Empty lines are allowed.
  • Comment lines start with #.

EXAMPLE

# This arguments file contains two arguments

# Ruby5
http://feeds.feedburner.com/Ruby5

# Software Engineering Radio
http://feeds.feedburner.com/se-radio
